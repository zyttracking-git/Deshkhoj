# Agent Instructions: DeshKhoj Rebuild Strategy

## Core Directive
We are rebuilding the DeshKhoj platform (a JustDial-like local business directory). 
**DO NOT USE OR MODIFY THE CURRENT/OLD CODE.** We are only using it to collect information and understand the database structure.

## Plan
1. **Frontend First**: We are building the new frontend first using Next.js, Tailwind CSS, and Framer Motion. 
2. **Backend Later**: Once the frontend is ready, we will build the backend using Node.js/Next.js API routes, PostgreSQL, and Prisma/TypeORM.
3. **Replacement**: Only after both the new frontend and backend are fully ready and tested, we will replace the old code entirely with the new one.

Please adhere strictly to this plan. Do not try to integrate the new frontend with the old PHP/MariaDB codebase.
